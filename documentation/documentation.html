2) HTML structure - top

  • Header - The header is always positioned at the top of the page. It consists on a div named "header” class, which encloses a div named class “logo” and an unordered list named “menu” (main menu). After if you are show source of home page, you will see slider section.
  • Slider – The slider section is always positioned at bottom of the header and above of the main container. It consists on a div named “row” for the homepages.
  • Main content(Homepage) - A class "content_body" contains the whole content area. Upper 3 columns consist of 3 classes "three_col_left","three_col" and "three_col_right" .
  • Main content (other pages) – A class "content_body" contains the whole content area. Inside this, there is another div "box" which contains the left and right side contents.
  • Box– Content area. That area contains following parts.
    • Left content area – it consists of a class "box_left"
    • Right sidebar – it consists of a class "box_right"
  • Bottom(homepage) – There is a small slider at the bottom part of the homepage. It consists of a class "advert2".
  • Footer - The footer area is always positioned at the bottom of the bottom area and above of the extra2 area. It consists on a div named “footer” class
  • Extra– This contains the copyright text and the social bookmarks

3) CSS files & Structure - top

The template contains following CSS files:
  • default.css – This is the main style sheet file.
  • style.css – It contains the buttons and link color styles.
  • portfolio.css – It styles filterable portfolio.
  • mosaic.css -It styles hover effect of the filterable portfolio
  • piecemaker.css - It styles the piecemaker slider.
  • nivo-slider.css - It structures the nivo slider.
  • pascal.css - It styles the nivo slider.
  • accordion.css - It styles the accordion slider.
  • prettyPhoto.css - It styles the modal window displayed when looking at the portfolio or content images.
  • coda-slider-2.0.css - It styles the content slider in product details page.
  • jquery.jcarousel.css – It styles the homepage content carousel slider.
The last file (default.css) contains all of the specific styling for pages. The file is separated into sections:
  • Common part (containing styles for basic elements p,a,body)
  • Heading Style (containing styles for the text and styles for the from H1 to H6)
  • List Styles (containing styles for the lists)
  • Notice Styles (containing styles for the notice boxes)
  • CORE STRUCTURE (containing styles for the page stucture)
  • Header style (containing the styles for the header, logo)
  • Navigation menu style (containing the styles for the navigation menu)
  • About us styles (containing the styles for about page)
  • Sidebar style (containing the styles for the sidebar)
  • Other part styles (containing the styles for the inside of content): Products, Portfolio, blog, service, Pricing, error page, Testimonials, Typography

6) PHP files - top

I've used the one PHP file named mail.php. That file used to send your contact information to your mail.

Adding Your Email Address

To add your own email address to the form, open the file "mail.php" and at the very top of the file replace "info@domain.com" with your own email address. Be sure not to delete the quotes, semi-colon or other code when making this change.

$webemail="info@domain.com"; // Replace with your email address


Problems Sending Emails

If you experience trouble sending emails you should verify the following:

1. Make sure the address entered in "mail.php" is correct.
2. Verify that PHP is installed on the server your site is running on.
3. If necessary contact your hosting provider to ensure they do not need to activate something.


7) Customizing - top

Enter your site name – You can set the site name in the title tag of the each html files on fifth line.

<title>Your site name</title>


Set Logo - You can use any image on your logo. Default image is set to 202px width and 59px height. If you need to adjust positioning, just edit the top margin value of logo class.

Adding Image - Add a normal image, just use the "img" tag.

Full-width - Use .full_width class to create f ull width page.

Customize colors - go to style.css and change color of the links on 5th line. for different color button try out changing the colors where /*change color here */ is mentioned.

Changing default background pattern of your site is to easy. You need to add class named your preffered pattern (default-pat,head-pat1, head-pat2 ...).

Before publishing

To keep your site as lightweight as possible, remember to do the following before publishing:

Remove comments from code

This can be achieved by a simple search/replace with the following regular expression :<!--[^\[].*?-->\s*

Minimise CSS and JS

To reduce bandwith and server load, combine and compress css and js files:

CSS files http://www.cleancss.com/

JS files http://yui.2clics.net/

Compress png files
Use pngcrush or an online tool (ie : http://www.r1ch.net/stuff/gif2png/webinterface/pngcrush.cgi) to reduce files size.
